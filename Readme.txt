◎はじめに

ほぼ自分用みたいな部分がありますが、あれば良いなみたいな機能はまとめています
初期機能の追加希望があればissuesまでどうぞ。



◎環境構築

まずこちらのテンプレートをビルドする上でVisual Studio 2022とDxLibが必要です
Visual Studioは当然持っていると思いますが、一応ダウンロードできる公式サイトのリンクを貼っておきます
https://visualstudio.microsoft.com/ja/

持っていなかった人のために一応ビルド環境の構築も説明しておきます
Visual Studio Installerがインストールできたら
"C++によるデスクトップ開発"の項目を選択してインストールしてください

そしたらあとは少し待ってインストールが完了すればビルド環境が整います

Visual Studioがインストールできたら次にDxLibをダウンロードしましょう
https://dxlib.xsrv.jp/index.html

そしたらダウンロードしたファイルを解凍し
以下のように設置してください

C:/DxLib_VC/プロジェクトに追加すべきファイル_VC用

これでこのテンプレートをビルドする環境が整いました



◎コードスニペット(code snippet)

では次に実際にコードを書いていく作業になるのですが、オブジェクトが多くなれば多くなるほど書くものが多いです
この書くものをなるべく減らせるように、コードスニペットを作りました

[ツール]->[コードスニペットマネージャー]から
snippetフォルダーの中にあるsnippetファイルをすべてインポートすれば、他のコードスニペットと同じ感じで使えます

ほぼこのテンプレート用なので他のプロジェクトで使えるかはわかりませんが
